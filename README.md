# execview

This is an extension which provides an intutive view of ELF binary.

## Features

Shows the segment view of the binary
Shows the sections view of the binary



## Extension Settings

Include if your extension adds any VS Code settings through the `contributes.configuration` extension point.

For example:

This extension contributes the following settings:

* `execview.enable`: Enable/disable this extension.
